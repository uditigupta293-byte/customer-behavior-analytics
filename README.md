# 🛍️ Customer Behavior Analytics | End-to-End Retail Data Analysis

## 📌 Project Overview

This project is an end-to-end customer shopping behavior analysis aimed at uncovering key business insights related to customer purchasing patterns, spending behavior, subscriptions, discounts, and product preferences.

The project follows a complete analytics workflow using Python, MySQL, and Power BI to transform raw customer transaction data into actionable business insights.

---

## 🎯 Business Problem

The objective of this project was to answer critical business questions such as:

- Which customer segments generate the highest revenue?
- Are repeat customers more likely to subscribe?
- Which products depend heavily on discounts?
- Which age groups contribute the most revenue?
- What are the top-performing products within each category?
- How do customer demographics impact purchasing behavior?
- Which categories and seasons drive maximum sales?

## 📈 Key Business Insights

- Young and middle-aged customers contributed the highest revenue.
- Subscription customers exhibited stronger purchasing behavior.
- Certain product categories showed high dependency on discounts.
- Repeat customers demonstrated higher loyalty and spending patterns.
- Seasonal trends significantly impacted customer purchasing behavior.
